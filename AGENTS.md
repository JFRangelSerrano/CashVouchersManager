# Agent Rules

- Cada función, método o procedimiento debe incluir un bloque de comentario en idioma inglés encima del mismo siguiendo el estandar del lenguaje, con la finalidad de explicar que hace dicho bloque de código, cada uno de sus parámetros y el valor retornado, en caso de retornar algo
- Todas las variables locales y parámetros de métodos deben escribirse en CamelCase.
- Para todo lo “público” o estructural (Clases, Métodos, Interfáces, Tipos enumerados, etc. ), debe usarse PascalCase
